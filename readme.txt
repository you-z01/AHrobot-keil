
有符号 int  4字节（32位）   能表示  -32767  到 32767
无符号 int  4字节（32位）   能表示  0       到 65535
int8_t      使用 signed char    定义，占用1字节，可以表示 -128          到 127
uint8_t     使用 unsigned char  定义，占用1字节，可以表示 0             到 255
int16_t     使用 signed short   定义，占用2字节，可以表示 -32768        到 32767
uint16_t    使用 unsigned short 定义，占用2字节，可以表示 0             到 65535
int32_t     使用 signed int     定义，占用4字节，可以表示 -2147483648   到 2147483647
uint32_t    使用 unsigned int   定义，占用4字节，可以表示 0             到 4294967295
